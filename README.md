## Project Overview
Project Overview
This is an Automated API test suite for Zedu Chat staging API `https://api.staging.zedu.chat/api/v1/` built with **JavaScript**, **Mocha**, and **Faker.js**. The project requires interns to build a structured API automation project against the Zedu platform. Interns are expecteed to automate the process and be able to hand the project to another engineer, and they should be able to clone it, follow README, and run your entire suite without asking you a single question.

## Prerequisites
- Node.js >= 18.0.0
- npm >= 9.0.0
- @faker-js/faker": "^9.3.0"
- axios: "^1.15.2"
- dotenv: "^16.6.1"
- mocha: "^11.7.5"
- jest": 29.7.0
- supertest": 7.0.0

### Setup Instructions
1. Clone the repository
git clone https://github.com/sysadmon/apiproject
cd apiproject

### How to run the full test suites

1. open google doc file from the URL https://docs.google.com/document/d/1Zt5c4Wep3BwmROjITcg-0zDFxhk1AVnwDpVj5EyySDw/edit?tab=t.0

2. Copy the entire code and paste in a new file with the name .env and save in your base folder

3. run npm test
